# Semantic Search Engine using Vector Database

This project implements a semantic search engine using vector embeddings.

Users can query documents using natural language and retrieve the most relevant results using similarity search.

## Tech Stack

Python
FastAPI
GEMINI Embeddings
PINECONE

## Features

Semantic search using embeddings
Vector database indexing
FastAPI REST API
Top-K similarity retrieval

## Run

pip install -r requirements.txt

uvicorn app.main:app --reload
